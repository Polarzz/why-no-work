# why-no-work
An esoteric programming language that has no use and serves no purpose other than to take the piss out of those idiots.

<h1> Instructions </h1>

`why -> value += 1` <br>

`work -> value -= 1` <br>

`no -> value += 10` <br>

`not -> value -= 10` <br>

`:( -> value *= 2` <br>

`:) -> value /= 2` <br>

`pls -> square value` <br>

`! -> output an ascii character with value [value]` <br>

`? -> output value as an integer` <br>

`help -> reset value` <br>
